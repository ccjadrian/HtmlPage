K2 Designer

创建一个直观的用户界面是困难和繁琐的，每次开发一个界面都需要重新编写代码或者是手动生成相关的代码。

现在...

有了K2 smartforms基于Web的流程设计器（也称为K2 designer），它是无代码的，即便没有技术背景的流程设计者也可以简单创造复杂的流程模式。当你开始设计一支流程时，流程设计向导就打开了。K2 smartforms提供了视图、表单、控制、规则，这些对象的配置都是向导驱动的，向导通过每一个配置步骤引导着K2 designer流程设计器。

可复用的拖拽控件紧密的流程集成方便整合多系统的数据灵活的可扩展能力

K2 Designer，能让用户快速、简单、无代码开发并自动生产业务流程、表单，整合系统数据，开箱即用。由于K2 smartforms是基于SmartObject作为其数据对象载体，集成性强大，甚至可将异构系统的数据直接用表单展现出来。

K2 for Visual Studio

设计器的集大成者，具备丰富的代码编辑能力，适合开发人员使用。

该工具主要功能特性包括：

图形化流程设计视图

允许用户以拖拽方式设计流程，使用符合WFMC标准的流程描述元素(如，流程步骤使用Activity元素等)进行流程设计，设计出的流程图简洁明了，能直观反映出实际流程走向。

无缝集成的编译、调试功能

Output 能显示整个流程编译过程，Error List 在流程编译出错时能主动监测到，断点设计允许程序员单步调试流程逻辑，Solutions视图用来管理流程项目，流程后台代码融合到智能代码编写环境中。

配置式的流程逻辑

通过流程逻辑规则，简化了流程设计的复杂度，借助这些流程规则能轻松实现复杂的审批路由、并串混合流程、任务超时提醒或转派、父子流程关联等流程逻辑。

提供丰富的流程向导模板

流程设计人员可以利用流程向导模板轻松开发出符合企业实际情况的流程，开发人员也可自定义流程向导模板于各个流程中复用。
